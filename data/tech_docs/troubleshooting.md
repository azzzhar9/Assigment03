# Technical Troubleshooting Guide

## Common Technical Issues and Solutions

### Internet Connectivity

**Problem**: Cannot connect to internet or slow connection

**Solutions**:
1. Check physical connections (Ethernet cable, Wi-Fi)
2. Restart router/modem
3. Check if other devices can connect
4. Disable and re-enable network adapter
5. Run network troubleshooter (Windows) or Network Diagnostics (Mac)
6. Contact ISP if issue persists

**For Remote Workers**:
- Test home internet speed (speedtest.net)
- Move closer to Wi-Fi router
- Use wired connection if possible
- Check VPN connection status

### Email Not Working

**Problem**: Cannot send or receive emails

**Solutions**:
1. Check internet connection
2. Verify email server status (status.company.com)
3. Clear browser cache if using webmail
4. Restart email client (Outlook, Mail app)
5. Check email account settings
6. Verify password hasn't expired
7. Check if mailbox is full (delete old emails)

**Outlook Specific**:
- Run Outlook in safe mode
- Repair Outlook profile
- Recreate email account

### Slow Computer Performance

**Problem**: Computer running slowly

**Solutions**:
1. Restart computer
2. Close unnecessary applications
3. Check Task Manager (Windows) or Activity Monitor (Mac) for resource usage
4. Clear browser cache and cookies
5. Run disk cleanup
6. Check available disk space (need at least 10% free)
7. Update software and drivers
8. Scan for malware
9. Consider hardware upgrade if consistently slow

### Printer Not Working

**Problem**: Cannot print or printer errors

**Solutions**:
1. Check printer is powered on
2. Verify paper and toner levels
3. Check for paper jams
4. Restart printer
5. Check printer queue for stuck jobs
6. Reinstall printer driver
7. Check network connection (for network printers)
8. Verify printer is set as default

### VPN Connection Issues

**Problem**: Cannot connect to VPN

**Solutions**:
1. Check internet connection
2. Verify VPN credentials
3. Update VPN client software
4. Check firewall settings
5. Try different VPN server location
6. Restart VPN client
7. Clear VPN cache
8. Check if VPN service is operational

### Software Crashes

**Problem**: Application keeps crashing

**Solutions**:
1. Restart the application
2. Restart computer
3. Update the application
4. Check for software conflicts
5. Reinstall the application
6. Check system requirements
7. Run application in compatibility mode
8. Check application logs for errors

### File Access Issues

**Problem**: Cannot access files or folders

**Solutions**:
1. Check file/folder permissions
2. Verify network connection (for network drives)
3. Check if file is locked by another user
4. Verify file path is correct
5. Check disk space
6. Run disk check utility
7. Contact file owner for permissions

### Audio/Video Issues

**Problem**: No sound or video not working

**Solutions**:
1. Check volume settings
2. Verify audio/video device is connected
3. Check device drivers
4. Restart application
5. Test with different application
6. Check system sound settings
7. Update audio/video drivers

### Password Issues

**Problem**: Cannot log in or password not working

**Solutions**:
1. Use self-service password reset
2. Check Caps Lock is off
3. Verify correct username format
4. Clear browser cache and cookies
5. Try different browser
6. Contact IT for password reset

## When to Contact IT Support

Contact IT immediately for:
- Security incidents (suspected malware, phishing)
- Data loss or corruption
- System-wide outages
- Unauthorized access attempts
- Hardware failures
- Critical business application failures

For non-urgent issues, submit a ticket through the IT Support Portal.
